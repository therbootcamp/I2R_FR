2 - kMeet

Nous tiendrons le cours via le logiciel kMeet.

Si vous n'avez pas encore de compte, veuillez vous inscrire (gratuitement) sur https://www.infomaniak.com/fr/kmeet. S'il vous plaît aussi installer l'application desktop. Telechargez l'application disponible sur https://www.infomaniak.com/fr/applications/telecharger-kmeet et effectuez l'installation. Si une installation du logiciel n'est pas possible, il est également possible de rejoindre Zoom via le navigateur avec https://kmeet.infomaniak.com/

Pendant l'événement, nous sommes constamment dans un canal principal, avec la possibilité de s'absenter pendant les exercices dans les salles de sous-groupes. Pendant toute la durée de l'événement, nous vous demandons de laisser la vidéo allumée et le microphone éteint par défaut. Si vous le souhaitez, vous pouvez définir un arrière-plan dans le menu des paramètres de l'application pour protéger votre vie privée. Pour poser des questions ou contribuer oralement pendant l'événement, il suffit d'allumer temporairement votre microphone (avec barre d'espace ou souris).

Nous commençons ponctuellement à 13h00. Veuillez arriver à la réunion kMeet 10 minutes à l'avance afin de pouvoir éliminer les difficultés techniques qui pourraient survenir (voir liens mentionnés auparavent).
